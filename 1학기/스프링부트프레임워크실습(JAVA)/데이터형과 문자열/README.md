<h1>데이터형</h1>
<ul>
  <li>변수나 상수의 종류를 데이터형이라고 함</li>
  <li>가장 많이 사용되는 기본 데이터형</li>
  <p>정수, 실수, 문자, 불(Boolean), 문자열</p>
</ul>
